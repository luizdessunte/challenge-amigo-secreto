      Sobre o Amigo Secreto:

      Introdução

      O Amigo Secreto é uma tradição seguida para aproximar as pessoas, tornando as celebrações, sejam elas de fim de ano, ou algum evento ainda mais divertidas. Criamos este site para o processo de sorteio ficar mais dinâmico e rápido, garantindo que cada participante receba um nome de forma completamente aleatória.

      Como Funciona

      Para começar a utilizar a funcionalidade de sorteio, basta digitar os nomes dos participantes no campo "Digite um nome" e clicar em "Adicionar". Pedimos para que você adicione no mínimo 3 nomes para iniciar o sorteio. O sistema embaralha os nomes e garante que ninguém sorteie seu próprio nome, exibindo o resultado completo da pessoa que retirou o nome de seu amigo.

      Objetivos

      Este site foi desenvolvido com o intuito de facilitar a organização do Amigo Secreto, promovendo uma experiência dinâmica para todos os participantes. Através de uma interface intuitiva, o sistema busca garantir transparência e agilidade no sorteio.

      Detalhes Técnicos

      Desenvolvido utilizando HTML, CSS e JavaScript, o site emprega técnicas de embaralhamento de arrays para assegurar a aleatoriedade do sorteio. Todo o design foi pensado para proporcionar uma experiência visual agradável e funcional, adaptando-se a diferentes dispositivos.
